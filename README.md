# Social
build social look like facebook
Social Network API
Đây là dự án xây dựng hệ thống mạng xã hội** với các chức năng cơ bản: đăng ký, đăng nhập, đăng bài viết, bình luận, tương tác (like), và quản lý người dùng. Dự án được xây dựng bằng **Spring Boot**.

# Công nghệ sử dụng

**Java**
- **Spring Boot**
- **Spring Data JPA**
- **MySQL *
- **Lombok**
- **JWT Authentication**
- **Maven**


## Tính năng chính
- Đăng ký tài khoản
- Đăng nhập bằng JWT
- Xem hồ sơ người dùng  

###  Bài viết
- Tạo bài viết
- Cập nhật / Xóa bài viết
- Xem danh sách bài viết

### Bình luận
- Thêm bình luận
- Xóa bình luận

###  Reaction (Like)
- Thả cảm xúc vào bài viết
- Thả cảm xúc vào bình luận
